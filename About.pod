This repository has several branches, that support several different (but
similar) CPAN distributions:

=over

=item C<xs>

L<Acme::Math::XS> is a plain old XS version.

=item C<eumm>

L<Alt::Acme::Math::XS::EUMM> is a version written using L<Inline::Module> and
L<Inline::C> that uses the plain old L<ExtUtils::MakeMaker> module framework.

=item C<dzil>

L<Alt::Acme::Math::XS::DistZilla> is a version that uses the L<Dist::Zilla>
module framework.

=item C<m-b>

L<Alt::Acme::Math::XS::ModuleBuild> is a version that uses the
L<Module::Build> module framework.

=item C<m-i>

L<Alt::Acme::Math::XS::ModuleInstall> is a version that uses the
L<Module::Install> module framework.

=item C<zild>

L<Alt::Acme::Math::XS::ZillaDist> is a version that uses the L<ZillaDist>
module framework.

=item C<cpp>

L<Alt::Acme::Math::XS::CPP> is a version written using L<Inline::Module> and
L<Inline::CPP> (C++) (and L<ExtUtils::MakeMaker>).

=back

The C<master> branch just has the code, doc and tests that should be shared by
every other branch.
